# DMV Helper Admin Panel

This is a powerful management tool built with **Flutter (Web & Desktop)** 
designed to maintain and scale the database for the **DMV Test Helper** application.

### ⚙️ Key Features:
*   **AI-Powered Content Engine:** Integrated with **Google AI Studio (Gemini API)** 
*    to automate the analysis of raw safety data and transform it into structured exam questions.
*   **Automated Localization:** The AI module handles real-time translation and cultural adaptation of questions across **8 different languages**.
*   **Database Acceleration:** Significantly reduces manual data entry by using AI to parse handbook excerpts into validated Firestore documents.
*   **Multi-State Synchronization:** Ensures all state-specific traffic laws are accurately reflected and updated in the global database.
*   **Role-Based Access Control:** Secure management system with Admin, Editor, and Viewer roles via **Firebase Auth**.

### 🛠 Tech Stack:
*   **Framework:** Flutter (Web & Windows Desktop)
*   **Backend:** Firebase (Firestore, Authentication, Cloud Functions, Remote Config)
*   **AI Integration:** Google Generative AI (Gemini SDK)
*   **Architecture:** Provider (State Management), Clean API Layer, Model-Driven Development. integration.
